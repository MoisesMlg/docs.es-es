## <a name="type-promotion"></a><span data-ttu-id="94b1e-161">Promoción de tipos</span><span class="sxs-lookup"><span data-stu-id="94b1e-161">Type Promotion</span></span>  

 <span data-ttu-id="94b1e-162">La clase <xref:System.Xml.Schema.XmlSchemaInference> examina los valores de atributo y elemento uno por uno.</span><span class="sxs-lookup"><span data-stu-id="94b1e-162">The <xref:System.Xml.Schema.XmlSchemaInference> class examines attribute and element values one at a time.</span></span> <span data-ttu-id="94b1e-163">A medida que se encuentran valores, se deduce el tipo sin signo más restrictivo.</span><span class="sxs-lookup"><span data-stu-id="94b1e-163">As values are encountered, the most restrictive, unsigned type is inferred.</span></span> <span data-ttu-id="94b1e-164">Si un tipo ha sido deducido para un atributo o elemento y se encuentra un nuevo valor que no coincide con el tipo deducido actual, el tipo deducido se promueve a un nuevo tipo que se aplica al tipo deducido actualmente y al nuevo valor.</span><span class="sxs-lookup"><span data-stu-id="94b1e-164">If a type has been inferred for an attribute or element, and a new value is encountered that does not match the currently inferred type, the inferred type is promoted to a new type that applies to both the currently inferred type and the new value.</span></span> <span data-ttu-id="94b1e-165">La clase <xref:System.Xml.Schema.XmlSchemaInference> considera valores anteriores cuando promueve el tipo deducido.</span><span class="sxs-lookup"><span data-stu-id="94b1e-165">The <xref:System.Xml.Schema.XmlSchemaInference> class does consider previous values when promoting the inferred type.</span></span>  
  
 <span data-ttu-id="94b1e-166">Por ejemplo, considere los siguientes fragmentos XML de dos documentos XML:</span><span class="sxs-lookup"><span data-stu-id="94b1e-166">For example, consider the following XML fragments from two XML documents:</span></span>

 <span data-ttu-id="94b1e-169">A continuación, considere el siguiente XML de dos documentos XML:</span><span class="sxs-lookup"><span data-stu-id="94b1e-169">Now, consider the following XML from two XML documents:</span></span>
