## <a name="example"></a><span data-ttu-id="91ba2-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="91ba2-107">Example</span></span>

```csharp  
// CS0206.cs

    public static int P  
    {  
        get  
        {  
            return 0;  
        }  
        set

        }  
    }

    }

    {  
        MyMethod(ref P);   // CS0206  
        // try the following line instead  
        // MyMethod(P);   // CS0206  
    }  
}  
```
