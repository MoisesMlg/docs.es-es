## <a name="using-static-type"></a><span data-ttu-id="74dcf-112">Uso de tipos estáticos</span><span class="sxs-lookup"><span data-stu-id="74dcf-112">Using static type</span></span>

```csharp

{
    static void Main()
    {
        WriteLine(Sqrt(3*3 + 4*4));
    }
}
```
