# <a name="icorprofilerobjectenumreset-method"></a>ICorProfilerObjectEnum::Reset (Método)

Mueve el cursor de este enumerador a la posición inicial de la secuencia.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
HRESULT Reset ();  
```
