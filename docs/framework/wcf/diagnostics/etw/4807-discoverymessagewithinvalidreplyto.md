# <a name="4807---discoverymessagewithinvalidreplyto"></a>4807 - DiscoveryMessageWithInvalidReplyTo

## <a name="properties"></a>Propiedades  
  
|||  
|-|-|  
|ID|4807|  
|Palabras clave|Detección|  
|Nivel|Advertencia|  
|Canal|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descripción  

 Este evento se genera cuando se quita un mensaje de solicitud de detección porque tenía una dirección ReplyTo no válida.  
  
## <a name="message"></a>Message  

 Se quitó un mensaje de solicitud de detección con messageId='%1' porque tenía una dirección ReplyTo no válida.
