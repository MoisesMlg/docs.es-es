# <a name="62326---httphandlerpickedforurl"></a>62326 - HttpHandlerPickedForUrl

## <a name="properties"></a>Propiedades  
  
|||  
|-|-|  
|ID|62326|  
|Palabras clave|WebHost|  
|Nivel|Información|  
|Canal|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descripción  

 Este evento se genera cuando el controlador HTTP se ha seleccionado para atender todas las solicitudes realizadas a esta dirección URL.  
  
## <a name="message"></a>Message  

 La dirección URL '%1' hospeda un documento XAML con el tipo de elemento raíz '%2'. El tipo de controlador HTTP '%3' se ha seleccionado para atender todas las solicitudes realizadas a esta dirección URL.
