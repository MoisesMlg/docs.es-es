# <a name="securitynegotiationsuccess"></a>SecurityNegotiationSuccess

ID.: 171  
  
 Gravedad: informativo  
  
 Categoría: SecurityAudit  
  
## <a name="description"></a>Descripción  

 Este evento indica que la negociación de seguridad ha tenido éxito, haciendo una lista del nombre del servicio, acción, identidad del cliente, ActivityId y negociación.
