# <a name="protocolinitializationfailure"></a>ProtocolInitializationFailure

ID.: 142  
  
 Gravedad: error  
  
 Categoría: TransactionBridge  
  
## <a name="description"></a>Descripción  

 Este evento indica que el servicio de protocolo WS-Atomic Transaction  no se inicializó. Como resultado, la funcionalidad WS-AtomicTransaction está deshabilitada. El evento indica el identificador de protocolo, el nombre de protocolo, la excepción, el nombre del proceso y el identificador del proceso. .
