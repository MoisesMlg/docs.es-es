# <a name="coordinatorrecoverylogentrycreationfailure"></a>CoordinatorRecoveryLogEntryCreationFailure

ID.: 140  
  
 Gravedad: error  
  
 Categoría: TransactionBridge  
  
## <a name="description"></a>Descripción  

 Una entrada de registro de recuperación no se pudo generar para la inscripción del coordinador. Se anulará la transacción. El evento enumera el id. de transacción, el motivo, la excepción, y el nombre y el id. de proceso.
