# <a name="protocolrecoverybeginningfailure"></a>ProtocolRecoveryBeginningFailure

ID.: 144  
  
 Gravedad: error  
  
 Categoría: TransactionBridge  
  
## <a name="description"></a>Descripción  

 Este evento indica que se produjo un error en el protocolo de transacción WS-Atomic de MSDTC al comienzo de la recuperación. Como resultado, la funcionalidad WS-AtomicTransaction está deshabilitada. El evento indica el identificador de protocolo, el nombre de protocolo, la excepción, el nombre del proceso y el identificador del proceso.
