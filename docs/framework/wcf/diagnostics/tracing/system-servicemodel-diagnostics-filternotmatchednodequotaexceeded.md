System.ServiceModel.Diagnostics.FilterNotMatchedNodeQuotaExceeded  
  
## <a name="description"></a>Descripción  

 La evaluación del filtro del registro de mensajes frente al mensaje superó la cuota de nodo que se había establecido en el filtro.
